Laborationer

Laborationer är rekommenderade övningsuppgifter, som tar upp många viktiga koncept på ett förklarande vis. Studenter kan titta på planen för 2012 för handledningstider. Där syns också när respektive labb bör vara gjord, för att hålla kursens tempo.

Alla fem laborationer är publicerade:

  1. Enkel personlig webbsida
  2. Mer HTML-element och stilmallar
  3. Layout med CSS
  4. Interaktivitet med JavaScript
  5. Media Queries

Labbgrupper HT 2012

Gruppuppdelning är enkelt konstruerad:

  1. Efternamn vars första bokstav är A-K.
  2. Efternamn vars första bokstav är L-Ö.

Det betyder att grupperna nästan, men inte riktigt, motsvarar de i Introduktion till datavetenskap.

För alla labbtider utom vecka 38 finns två datorsalar bokade parallellt. Grupp 1 går till salen som står först på schemat, grupp 2 går till den andra. (Exempel för 11/9: Grupp 1: C119, Grupp 2: C121.)

Notera också att det går några fler individer per grupp än vad det finns datorer i många salar. Det löser sig om ett fåtal i varje sal har bärbar dator med, eller kan samarbeta.

Användning av allt material är fritt, om motsatsen inte anges. Länka gärna! Formellt utrycks detta med licensen Creative Commons Erkännande 2.5 Sverige.

Detta projekt är publicerat som öppen källkod. Du kan ladda hem webbplatsen i sin helhet, och får gärna föreslå ändringar och bidra med förbättringar.
