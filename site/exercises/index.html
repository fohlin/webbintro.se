Laborationer

Laborationer är rekommenderade övningsuppgifter, som tar upp många viktiga koncept på ett förklarande vis. Studenter kan titta på planen för 2014 för handledningstider. Där syns också när respektive labb bör vara gjord, för att hålla kursens tempo.

Laborationerna kommer publiceras under kursens gång

  1. Enkel personlig webbsida
  2. Mer HTML-element och stilmallar
  3. Layout med CSS
  4. Interaktivitet med JavaScript
  5. Media Queries

Labbgrupper HT 2014

Till varje laboration har vi två datorsalar. Finns det inte plats i den första sal ni går i, testa den andra på schemat - där bör det finnas plats då.

Notera också att det går några fler individer per grupp än vad det finns datorer i många salar. Det löser sig om ett fåtal i varje sal har bärbar dator med, eller kan samarbeta.

Schemat med salar för laborationerna hittar ni här

Användning av allt material är fritt, om motsatsen inte anges. Länka gärna! Formellt utrycks detta med licensen Creative Commons Erkännande 2.5 Sverige.

Detta projekt är publicerat som öppen källkod. Du kan ladda hem webbplatsen i sin helhet, och får gärna föreslå ändringar och bidra med förbättringar.

Webbplatsen drivs av Fredrik Ohlin och Anton Tibblin, som också ansvarar för kursen med samma namn, vid programmet Informationsarkitekt, Malmö högskola.
